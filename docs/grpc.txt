The gRPC documentation for mcsv_receiveLocation:

The request would contain userId, latitude and longitude.

To make a sample request, please have a look in the file in the modules folder.
modules/mcsv_receiveLocation/location-event-client-grpc.py

To create a successful response, following data could be used as an example:
    userId=5,
    latitude=-100,
    longitude=30
